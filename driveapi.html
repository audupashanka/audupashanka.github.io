Drive API module

create_permission(role, currdirId, dirs, users)

This function will grant permission to a file/folder based on the type of role and the type of users provided.

Parameters

  • role (str) – type of permissions (reader/writer/editor).

  • currdirID (str) – Directory under which the search for the dirs begins.

  • dirs (list) – directory files .

  • users (list) – List of email ids on whom permission can be applied.

Returns

None

downloadFile(filepath, destpath=None, currdirId='root')

This function will download a file from google drive to the path specified.

Parameters

  • filepath (str) – the directory path of a file/folder.

  • destpath (str) – the destination folder path for the downloaded file.

  • currdirID (str) – Directory under which the search for the dirs begins.

Returns

None

make_owner(currdirId, filepath, user)

This function will make the given user a pending owner for a file/folder. Pending user has to accept ownership.

Parameters

  • currdirID (str) – Directory under which the search for the dirs begins.

  • filepath (list) – the directory path of a file/folder.

  • user (list) – email id of the new owner.

Returns

None
